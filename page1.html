EMPTY ELEMENTS

EMPTY ELEMENTS are element thst do not have an element content an end tag.

the <br > tag defines a line break , and is an empty element without a closing tag:

code:

<p > this is <br/ > paragraph with a line break

Output

            this is 
            paragraph with a line break

A list of commonly used Empty Elements:

  1. <meta />
  2. <link />
  3. <img />
  4. <br />
  5. <hr />
  6. <input />

The practice in empty elements is to alwaysput a forward slash / sign before the greater than >sign.

in this way they are closed at their tags . theae tags are also called self closing tags.

HTML is NOT a case sensitive

HTML tags are not a case sensitive : <P > means the same as <p > .<H1 > means the same as <h1>


THE HTML standard does not require lowercase tags, but W3C recommends lowercase in HTML, and demands lowercase for stricter documents types like XHTML.

we will always use lowercase tag names
